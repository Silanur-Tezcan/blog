🤖 Makine Öğrenmesi Projesi

Verileri analiz edip tahmin yapan bir model oluştur!

🛠️ Gereksinimler

  • Python 3.x
  • `pandas` ve `numpy` ile veri işlemesi
  • `scikit-learn` ile model oluşturma
  • Matplotlib ile veri görselleştirme

💻 Örnek Kod

            import pandas as pd
            from sklearn.model_selection import train_test_split
            from sklearn.linear_model import LinearRegression
            
            # Örnek veri seti
            data = {'x': [1, 2, 3, 4, 5], 'y': [2, 4, 6, 8, 10]}
            df = pd.DataFrame(data)

            X = df[['x']]
            y = df['y']

            # Veri setini bölme
            X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

            # Model oluşturma ve eğitme
            model = LinearRegression()
            model.fit(X_train, y_train)

            # Tahmin yapma
            print("Tahmin:", model.predict([[6]]))

🚀 Nasıl Yapılır?

  1. Python ortamınızı hazırlayın (Anaconda veya VirtualEnv kullanabilirsiniz).
  2. Gerekli kütüphaneleri yükleyin (`pip install scikit-learn pandas numpy`).
  3. Veri setinizi oluşturun veya gerçek verileri kullanın.
  4. Makine öğrenmesi modelini eğitin ve tahminler yapın.
