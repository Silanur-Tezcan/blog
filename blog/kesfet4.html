📖 C# Not Defteri Projesi

Windows Forms veya Konsol ile kendi not defterinizi oluşturun!

🛠️ Gereksinimler

  • C# .NET Framework / .NET Core
  • Windows Forms veya Konsol Uygulaması
  • Dosya okuma & yazma (`StreamWriter`, `StreamReader`)

💻 Örnek Kod

            using System;
            using System.IO;

            class Program
            {
                static void Main()
                {
                    Console.WriteLine("📖 Not Defteri - C#");
                    Console.Write("Kaydetmek istediğiniz notu girin: ");
                    string not = Console.ReadLine();

                    File.AppendAllText("notlar.txt", not + Environment.NewLine);
                    Console.WriteLine("✅ Not başarıyla kaydedildi!");
                }
            }

🚀 Nasıl Yapılır?

  1. Yeni bir C# konsol projesi oluşturun.
  2. `StreamWriter` ve `StreamReader` ile dosya işlemleri yapın.
  3. Notları kullanıcıdan alıp `notlar.txt` içine kaydedin.
  4. Windows Forms ile UI ekleyerek görsel bir arayüz oluşturabilirsiniz.
