🖥️ C# Masaüstü Uygulaması

Windows Forms veya WPF kullanarak şık bir masaüstü uygulaması oluştur!

🛠️ Gereksinimler

  • C# .NET Framework / .NET Core
  • Windows Forms veya WPF
  • Kullanıcıdan veri alma ve kaydetme
  • Şık ve kullanışlı arayüz tasarımı

💻 Örnek Kod

            using System;
            using System.Windows.Forms;
            using System.IO;

            public class NotDefteri : Form
            {
                TextBox txtNot = new TextBox();
                Button btnKaydet = new Button();

                public NotDefteri()
                {
                    Text = "C# Masaüstü Not Defteri";
                    Size = new System.Drawing.Size(400, 300);

                    txtNot.Multiline = true;
                    txtNot.Size = new System.Drawing.Size(350, 150);
                    txtNot.Location = new System.Drawing.Point(20, 20);

                    btnKaydet.Text = "Kaydet";
                    btnKaydet.Location = new System.Drawing.Point(20, 200);
                    btnKaydet.Click += KaydetNot;

                    Controls.Add(txtNot);
                    Controls.Add(btnKaydet);
                }

                private void KaydetNot(object sender, EventArgs e)
                {
                    File.AppendAllText("notlar.txt", txtNot.Text + Environment.NewLine);
                    MessageBox.Show("✅ Not kaydedildi!");
                }
            }

            public static class Program
            {
                public static void Main()
                {
                    Application.Run(new NotDefteri());
                }
            }

🚀 Nasıl Yapılır?

  1. Visual Studio'da yeni bir **Windows Forms** projesi oluşturun.
  2. Arayüzü oluşturmak için `TextBox` ve `Button` bileşenleri ekleyin.
  3. Kullanıcının girdiği notları `StreamWriter` ile dosyaya kaydedin.
  4. Windows Forms veya WPF kullanarak görselliği geliştirin.
