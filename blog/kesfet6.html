🎮 C# ile Oyun Geliştirme

Unity ve C# kullanarak basit bir platform oyunu geliştir!

🛠️ Gereksinimler

  • Unity Engine
  • C# ile Oyun Mekanikleri
  • Fizik ve Çarpışma Yönetimi (`Rigidbody` ve `Collider` kullanımı)
  • Karakter Hareketi (`Input.GetAxis` ile kontrol sağlama)

💻 Örnek Kod (Basit Karakter Kontrolleri)

            using UnityEngine;

            public class PlayerController : MonoBehaviour
            {
                public float speed = 5f;
                public float jumpForce = 5f;
                private Rigidbody2D rb;

                void Start()
                {
                    rb = GetComponent();
                }

                void Update()
                {
                    float moveX = Input.GetAxis("Horizontal") * speed;
                    rb.velocity = new Vector2(moveX, rb.velocity.y);

                    if (Input.GetKeyDown(KeyCode.Space))
                    {
                        rb.velocity = new Vector2(rb.velocity.x, jumpForce);
                    }
                }
            }

🚀 Nasıl Yapılır?

  1. Unity'de yeni bir **2D Oyun Projesi** oluşturun.
  2. **Ana karakterinizi** sahneye ekleyin ve `Rigidbody2D`, `BoxCollider2D` gibi bileşenler ekleyin.
  3. Yukarıdaki C# kodu ile **karakter hareketini** ve **zıplamayı** kontrol edin.
  4. Arkaplan ve engeller ekleyerek **oyun dünyasını oluşturun**.
  5. Oyununuzu test edip geliştirmeye devam edin! 🚀
